## Usage
- Clone & run `npm install`
- To build run `gulp`
- To run browserSync while developing run `gulp dev`

## Theme Template
-  [Grayscale](http://startbootstrap.com/template-overviews/grayscale/) - [Start Bootstrap](http://startbootstrap.com/)
